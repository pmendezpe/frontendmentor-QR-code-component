# QR Code Component

## Description

This is a project based on the [QR Code Component](https://www.frontendmentor.io/challenges/qr-code-component) challenge from the **Frontend Mentor** platform. The goal of this project is to develop a simple and responsive card that contains a QR code and descriptive text using only **HTML** and **CSS**.

## Features

- **Responsive** design, following the **mobile-first** strategy.
- Use of **Flexbox** for content layout.
- Implementation of **Google Fonts** to enhance typography.
- Image optimization using **lazy loading**.
- Clean and semantic HTML5 code.

## Technologies Used

- **HTML5**: For the semantic structure of the content.
- **CSS3**: For styling and design.
